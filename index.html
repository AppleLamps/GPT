Help & FAQ

Getting Started

How do I start using YourLocalGPT?

1. Set your API key in Settings (required for using the app)

2. Choose a model or create a Custom GPT

3. Start a new chat and begin asking questions!

Where do I get an API key?

You can obtain API keys from:

  • OpenAI API key: OpenAI Dashboard
  • Google Gemini API key: Google AI Studio
  • X.AI API key: X.AI Platform

Features

What are Custom GPTs?

Custom GPTs are specialized AI assistants you can create with:

  • Custom instructions and behavior
  • Specific knowledge files
  • Capabilities like web search

What file types are supported?

You can upload:

  • .txt files - Plain text documents
  • .md files - Markdown formatted text
  • .pdf files - PDF documents (text content only)
  • .js files - JavaScript source code
  • .py files - Python scripts

Privacy & Storage

How is my data stored?

All data is stored locally in your browser:

  • API keys are encrypted and stored in local storage
  • Chat history is saved locally
  • Custom GPTs and their knowledge files are stored locally
  • No data is sent to our servers

How do I backup my data?

You can export your data using the Settings menu:

  • Export all chats and Custom GPTs
  • Export Custom GPTs only
  • Export chat history only

Troubleshooting

Common Issues

  • API Key Error: Ensure your API key is correctly entered and has sufficient credits
  • File Upload Issues: Check file size limits (max 1MB per file) and supported formats
  • Web Search Not Working: Only available with GPT-4o model

How do I clear my data?

You can:

  • Clear individual chats using the delete button
  • Clear all conversations using the sidebar option
  • Remove Custom GPTs individually or reset all settings
